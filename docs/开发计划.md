## Phase 3：深度教学

| 优先级 | 功能 | 预估工时 | 依赖 | 说明 |
|:------:|------|:--------:|:----:|------|
| P0 | **用户烹饪经验分级** | 3h | 持久化 | 首次弹窗询问「小白 / 会一点 / 熟练」；动态调整 system prompt 难度 |
| P1 | **渐进式技能解锁** | 6h | 经验分级 + 持久化 | 从简到繁推荐菜谱（番茄炒蛋→宫保鸡丁→水煮鱼），解锁后永久可用 |
| P1 | **对话主题/上下文优化** | 3h | 持久化 | 让 AI 记住"上次做过什么"、"哪些菜已经熟练"，持续优化推荐 |
| P2 | **食材分类/收藏** | 4h | 持久化 | 用户可保存常用食材组合，快速复用 |

## 依赖关系图

```
聊天记录持久化 ──→ 用户烹饪经验分级 ──→ 渐进式技能解锁
     │                                       │
     └──→ 对话主题/上下文优化 ←──────────────┘
```

- **Phase 2 优先**：持久化是所有个性化能力的基础，必须先做
- **Phase 3**：依赖 Phase 2 的持久化能力
- **Phase 4 语音**：可与 Phase 3 并行开发，无硬依赖
- **Phase 5**：依赖 Phase 3 的智能检测能力，建议靠后
